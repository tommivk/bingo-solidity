# bingo-solidity


## Contract

#### Install dependencies
```
npm install
```
#### Run tests

```
npx hardhat test
```
#### Compile contracts
```
npx hardhat compile
```
#### Compile contracts and update ABIs in the Frontend

```
npm run compile
```

#### Deploy contracts to Polygon Mumbai testnet

```
npx hardhat run scripts/deploy.ts --network mumbai
```


## Frontend 

#### Install dependencies
```
npm install
```
#### Run unit tests 
```
npm run test
```

#### Start in development mode
```
npm run dev
```



#### Build
```
npm run build
```

#### Start in production mode
```
npm run start
```
#### Build StoryBook
```
npm run build-storybook
```

#### Start StoryBook
```
npm run storybook
```
