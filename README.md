# bingo-solidity

## Frontend 

Install dependencies
```
npm install
```
Run unit tests 
```
npm run test
```

Start in development mode
```
npm run dev
```



Build
```
npm run build
```

Start in production mode
```
npm run start
```
Build StoryBook
```
npm run build-storybook
```

Start StoryBook
```
npm run storybook
```
